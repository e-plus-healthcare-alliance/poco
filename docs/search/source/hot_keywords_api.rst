Hot Keywords API
================

路径: /v1.6/public/hot-keywords/

HTTP方法: GET/POST

返回最热门的若干搜索关键词。

传入参数
---------

=============    ==========  ==========================================================   =============================================
参数名           是否必填    默认值                                                       描述                                         
=============    ==========  ==========================================================   =============================================
category_id      否                                                                       
amount           否          5                                                            返回结果数
api_key          是                                                                       分配给用户站点的api key
=============    ==========  ==========================================================   =============================================


返回结果
--------

==============    ===============================
名称               说明
==============    ===============================
hot_keywords      热门关键词
errors             错误信息。正常情况下为[]。
==============    ===============================

示例
------

请求::

    curl -X GET 'http://search.tuijianbao.net/api/v1.6/public/hot-keywords/' \
         -H 'Content-Type: application/json' \
         -d '{
             "api_key": "123456",
             "amount": 5
         }'

结果::

    { "hot_keywords": ["补肾", "六味地黄丸", "万艾可", "减肥", "同仁堂"],
      "errors": []
    }
